HyperText Markup Language (HTML)

HTML or HyperText Markup Language is the foundation of every website. It provides structure to web pages using elements and tags that define text, images, links, and other content.

Purpose and Structure

  • Elements and Tags: HTML uses tags to define content. Most come in pairsâ€”opening and closingâ€”surrounding their content.
  • Document Type Declaration: Every HTML file should start with <!DOCTYPE html> to indicate HTML5.
  • Basic Structure: Includes <head> for metadata and <body> for visible content.

Common HTML Elements

  • Headings: h1â€“h6 define titles.
  • Paragraphs: p defines blocks of text.
  • Links: a creates hyperlinks using href.
  • Images: img embeds pictures using src and alt.
  • Lists: ul and ol hold list items.

Attributes

Attributes add extra information to elements. They appear inside the opening tag, written as name="value" pairs. Common examples include href (for links) and src (for images).
