Git

Git is a powerful version control system that allows developers to track changes, collaborate efficiently, and maintain project history. In web development, it helps manage HTML, CSS, and other code files in a structured, organized way.

Core Concepts

  • Repository (Repo): Stores all files and history.
  • Working Directory: The files you're currently editing.
  • Staging Area (Index): Prepares changes before committing.
  • Commit: A snapshot of your project at a point in time.
  • Branch: A separate line of development.
  • Merge: Combines branches together.
  • Remote Repository: A hosted version (e.g., GitHub).
